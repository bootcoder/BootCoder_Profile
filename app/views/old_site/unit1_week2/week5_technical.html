Class Act

Classes are a great way to create similar objects in Ruby, or to group related functions together under one roof.
Making your own is super easy just:

class Dog
end

Fido = Dog.new

Pretty basic stuff really.

Each time you create a new Dog from the class they should receive certain pre-defined traits. Let's add some in using the initialize method.

class Dog
def initialize(name, breed)
@name = name
@breed = breed
end
end

Fido = Dog.new("Fido","Corgi")

The @ symbol in front of the variables denotes it as an instance variable.

An instance variable is available across functions within a class.

Lastly, let's throw in a method to demonstate class functionality.

class Dog
 def initialize(name, breed)
  @name = name
  @breed = breed
 end
 def speak
  puts "#{@name} says Woof Woof"
 end
end
Fido = Dog.new("Fido","Corgi")
Fido.speak
=> Fido says Woof Woof
