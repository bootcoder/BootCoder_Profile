Enumerable#group_by

Ruby's group_by mixin is a handy dandy thing to have around. Lets think on a grand scale. What if you had an array with the telephone numbers of all of S.F. But, lets say you wanted to sort them by prefix.

You can use group_by to easily accomplish this task. Producing the following hash made up of arrays for each prefix

nums = %w[554-323-7654 534-323-7654 554-323-7654
524-333-7654 524-323-7454 554-323-7054 554-323-7454
554-323-7614 524-323-7654]

nums.group_by { |n| n[0..2] }

{
"554"=>["554-323-7654", "554-323-7654", "554-323-7054", "554-323-7454", "554-323-7614"],
"534"=>["534-323-7654"],
"524"=>["524-333-7654", "524-323-7454", "524-323-7654"]
}
