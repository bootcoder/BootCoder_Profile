Week 2 - Why is JavaScript so awesome for web development.

I hardly have the bon-a-fides to comment expertly on the matter. But in so far as I can see

Javascript's main claim to fame is that it is the one truly global web development language,

Because it runs on all modern browsers. One could easily say that it is the best

language by offering the fact that there are more Javascript posts

on Github and StackOverflow than for any other language.

This in turn correlates to a very high degree of on-line resources.

The more resources, the more you know.

But, knowing is only...

The other half is Internet Explorer, seriously.
