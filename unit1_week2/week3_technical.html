Array vs. Hash

Arrays

Think of an array as a bucket you can throw things in. You can put whatever you want in the
bucket. Strings, numbers, booleans, objects etc... The order you add items is the order they are in unless you change it.

Arrays are used to contain groups of data,

Then allow you to work through that data to many useful ends.

Hashes

Think of Hashes as a being like a phonebook. Each name has a number.
Similarly, hashes contain groups of information. Each data set consisits of
two properties. The key and the value. Like age => 32
